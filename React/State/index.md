# State

## State是什么？

State是状态。

## State有什么用？

State决定页面的`UI渲染内容`与页面的`行为`。

## React如何管理状态？

React的状态有如下几大类：

- 全局状态
    全局状态一般通过`useContext`和`Redux`管理。

- 组件内部状态
    组件内部的状态一般通过`useState`和`useReducer`管理。
